This article demonstrates how to build and tune a PID controller for an inverted pendulum in Simulink using the Multibody library without requiring mathematical formulas or manual calculations. The system uses two control loops—one for stabilizing the pendulum in the upright position and one for controlling cart position—with automatic tuning via Simulink's built-in tools. Additional features like a Kalman filter for noise reduction and a swing-up mechanism are included to make the simulation more realistic and robust.
