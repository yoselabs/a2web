Based on the article, **framework-defined infrastructure (FdI)** is:

Framework-defined infrastructure is an evolution of Infrastructure as Code where a build-time program automatically parses framework source code, understands the developer's intent, and generates the necessary IaC configuration to provision infrastructure. This eliminates manual infrastructure configuration by having the framework itself dictate production behavior—mapping framework concepts (like Next.js routes or serverless functions) directly to appropriate cloud infrastructure without explicit declaration.
