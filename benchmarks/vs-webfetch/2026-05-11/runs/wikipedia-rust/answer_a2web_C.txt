**Original Designer and First Release:**
Rust was originally created by **Graydon Hoare** in 2006 as a personal project while working at Mozilla. The first public release, **Rust 0.1**, became available on **January 20, 2012** for Windows, Linux, and macOS.

**Memory-Safety Mechanism:**
Rust is known for its **borrow checker** as its primary memory-safety mechanism. Rather than using a conventional garbage collector, Rust enforces memory safety at compile time through the borrow checker, which:

- Tracks object lifetimes of references at compile time
- Prevents memory safety errors and data races through an ownership system
- Ensures that each value has exactly one owner
- Allows values to be borrowed temporarily (through references) before being returned to the owner
- Prevents the creation and use of dangling pointers

The ownership system is a core feature that allows Rust to guarantee memory safety without runtime overhead from garbage collection.
