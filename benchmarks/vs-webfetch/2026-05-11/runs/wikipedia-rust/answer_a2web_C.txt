**Designer and Release Year:**
Graydon Hoare, a Mozilla software developer, originally created Rust in 2006. The first public release, Rust 0.1, came out on January 20, 2012. The first stable release, Rust 1.0, was published in May 2015.

**Memory-Safety Mechanism:**
Rust is known for its **borrow checker**. This is a compile-time mechanism that enforces memory safety without requiring a conventional garbage collector. The borrow checker tracks the object lifetime of references at compile time, preventing memory safety errors and data races. As the content states, "Rust enforces memory safety (i.e., that all references point to valid memory) without a conventional garbage collector; instead, memory safety errors and data races are prevented by the 'borrow checker', which tracks the object lifetime of references at compile time."
