**Original Designer and First Release:**
Rust was created by **Graydon Hoare** in 2006 as a personal project while working at Mozilla. The **first public release (Rust 0.1) was on January 20, 2012** for Windows, Linux, and macOS.

**Memory-Safety Mechanism:**
Rust is known for its **"borrow checker"** as its primary memory-safety mechanism. Rather than using a conventional garbage collector, Rust enforces memory safety through compile-time tracking of object lifetimes via the borrow checker. This system prevents memory safety errors and data races by ensuring:

- All references point to valid memory
- Each value has exactly one owner at compile time
- Values can be moved between owners or temporarily borrowed
- The compiler validates that references are only used where they're valid

This approach allows Rust to guarantee memory safety without the runtime overhead of garbage collection, making it both safe and performant.
