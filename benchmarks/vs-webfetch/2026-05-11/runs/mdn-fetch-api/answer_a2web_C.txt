The Fetch API provides an interface for fetching resources (including across the network) and is a more powerful and flexible replacement for `XMLHttpRequest`.

The primary entry point is the **`fetch()` method**, which is a global method available in both `Window` and `Worker` contexts. The `fetch()` method takes a mandatory argument (the path to the resource) and optionally an `init` options object as a second argument. It returns a `Promise` that resolves to a `Response` object as soon as the server responds with headers—even if the response is an HTTP error status.
